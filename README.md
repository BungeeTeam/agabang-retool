## 주요 디렉토리 설명

### apps/
Retool 애플리케이션들이 저장된 디렉토리입니다. 각 앱은 브랜드별 또는 기능별로 구성되어 있습니다.

#### 브랜드별 앱
- **01. EttoiApp**: 에뜨와 브랜드 관련 앱
  - 인사이트, 리오더, 아이템맵, 판매 현황, 입출고현황, 시즌기획 리뷰, 설정 관리 등
- **02. AgabangApp**: 아가방 브랜드 관련 앱
  - 인사이트, 아이템맵, 판매현황, 입고현황, 시즌기획 리뷰, 설정 관리 등
- **03. DisneyApp**: 디즈니 브랜드 관련 앱
  - 인사이트, 리오더, 아이템맵, 판매현황, 입고현황, 시즌기획 리뷰, 설정 관리 등

#### 기능별 앱
- **04. 발주최적화**: 발주 최적화 관련 앱, 에뜨와/아가방/디즈니 모두에게 페이지별로 할당
- **11. 영업**: 영업 관련 앱들
  - 영업 현황, 영업MD 분석, 매장 정보 관리 등
- **21. 버그 제보_개선 요청**: 버그 제보 및 개선 요청 관리 앱
- **99. Module**: 여러 앱에서 공통으로 사용되는 모듈들
  - 인사이트, 입고현황, 시즌기획 리뷰, 아이템 상세, 리오더, 매출 통계 테이블 등
- **archive**: 더 이상 사용하지 않는 아카이브된 앱들

### docs/
프로젝트 문서가 저장된 디렉토리입니다. 각 앱의 데이터 플로우 문서, 지식 베이스 등이 포함되어 있습니다.

### queries/
여러 앱에서 공통으로 사용되는 쿼리들이 저장된 디렉토리입니다.
- 시즌 관련 쿼리 (get_latest_seasons, season_list 등)
- 스타일 관련 쿼리 (style_season_code, style_year_code 등)
- 기타 공통 쿼리들
